## 初筛方法

本轮不是只做关键词匹配，而是同时判断：

1. 产品是否实质落在 AI Agent、大模型 API/开发者平台、Context Engineering/Memory/MCP/Tool Use、新型人机交互、AI 原生生产力、Coding Agent/Developer Tools。
2. 官网或官方文档是否能说明真实机制，而不是只有泛化营销页。
3. 产品是否对 agent 工具调用、上下文工程、可观测性、评估、安全执行、桌面自动化、协作界面或垂直 agent 工作流有启发。
4. 是否存在降权因素，例如普通套壳、轻量效率工具、非 AI/agent 核心、信息不足、或只是传统产品的营销改名。

评分规则：相关度 0-5，产品机制新颖度 0-5，对 AI 产品设计的启发 0-5，市场信号 0-3，总分 18；只保留总分 >= 11 的产品。

## 今天最值得关注的 3 个产品

### 1. Firecrawl Research Index - 16/18

- Product Hunt：<https://www.producthunt.com/products/extract-by-firecrawl>
- 官方来源：<https://docs.firecrawl.dev/features/research>、<https://firecrawl.dev/blog/research-index-launch>
- 核实信息：官方文档称它是面向 scientific/engineering research agents 的专用索引，支持论文搜索、论文元数据与全文片段检索、相似/引用/参考文献扩展，以及研究相关 GitHub issues、PR、README 检索，并可通过 API、CLI、MCP、SDK 接入。
- 为什么值得关注：它把 research agent 的关键能力从普通网页搜索推进到“论文检索 + 证据片段验证 + 代码实现线索”的闭环，对 deep research agent、科研 coding agent 和可追溯答案设计都有直接启发。
- 评分：相关度 5，机制新颖度 5，设计启发 5，市场信号 1。

### 2. Unreal Engine 5.8 - 16/18

- Product Hunt：<https://www.producthunt.com/products/unreal-engine>
- 官方来源：<https://www.unrealengine.com/news/unreal-engine-5-8-is-now-available>
- 核实信息：Epic 官方发布说明称 Unreal Engine 5.8 新增 Experimental MCP plugin，使 LLM 系统可连接并理解引擎和项目，帮助构建 assets/systems、扩展引擎能力、执行测试和优化任务，并内置暴露 Blueprints、assets、levels、materials、meshes 等核心系统，开发者可继续扩展。
- 为什么值得关注：这是大型专业创作软件把内部对象模型开放给 agent 的标志性案例。MCP 不再只是开发者工具生态的连接层，也开始成为复杂创作环境的 agent control surface。
- 评分：相关度 5，机制新颖度 4，设计启发 5，市场信号 2。

### 3. API to MCP - 15/18

- Product Hunt：<https://www.producthunt.com/products/api-to-mcp>
- 官网：<https://apitomcp.ai/>
- 核实信息：官网显示它可把 REST/GraphQL、SaaS 与内部业务 API 转成 hosted remote HTTP MCP servers，支持 OAuth、API key、Bearer、Basic Auth、workflow tools、encrypted credentials、JMESPath output mapping，并提供 Visual Builder 与 Agent Builder 两条创建路径。
- 为什么值得关注：MCP 的落地难点是把真实 API 安全、可控、可测试地暴露给 agent。API to MCP 直接切 agent tool surface 的生产化基础设施，对企业内部 API agent-ready 很有参考价值。
- 评分：相关度 5，机制新颖度 4，设计启发 5，市场信号 1。
